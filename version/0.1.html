FFF Flavored Frontmatter
Version 0.1 #

Published: 2022-07-18

The first version of FFF Flavored Frontmatter, inherited most of the variables from Urara.

If they have a corresponding Microformats class name, it will be written at the end of the description.

Base #

Base variables: It has been implemented for some time and is more commonly used.

title #

  • type: string
  • description: The title of the post. p-name
title: 'Title'

title: 'Title'

summary #

  • type: string
  • description: summary[1] conveys the main point of the post. p-summary
summary: 'Lorem ipsum dolor sit amet, consectetur adipiscing elit.'

summary: 'Lorem ipsum dolor sit amet, consectetur adipiscing elit.'

created #

  • type: string
  • description: The created[2] date of the post, in any string - specified by the corresponding implementation.
created: 2022-07-18

created: 2022-07-18

updated #

  • type: string
  • description: The updated[3] date of the post, in any string - specified by the corresponding implementation. dt-updated
updated: 2022-07-18

updated: 2022-07-18

published #

  • type: string
  • description: The published[4] date of the post, in any string - specified by the corresponding implementation. dt-published
updated: 2022-07-18

updated: 2022-07-18

photo #

  • type: string | string[] (optional)
  • description: Link to an image. u-photo[5] or u-featured[6], also support multi-photo[7].
photo: '/images/foo.avif'
# or
photo:
- '/images/foo.avif'
- '/images/bar.avif'

photo: '/images/foo.avif'
# or
photo:
- '/images/foo.avif'
- '/images/bar.avif'

alt #

  • type: string | string[] (optional)
  • description: Optional image alternative, also available as figcaption.
photo: '/images/foo.avif'
alt: 'lorem ipsum'
# or
photo:
- '/images/foo.avif'
- '/images/bar.avif'
alt:
- 'lorem'
- 'ipsum'

photo: '/images/foo.avif'
alt: 'lorem ipsum'
# or
photo:
- '/images/foo.avif'
- '/images/bar.avif'
alt:
- 'lorem'
- 'ipsum'

tags #

  • type: string[]
  • description: Tags are provided in array. also as an alternative to categories like as u-category
tags:
  - foo
  - bar

tags:
  - foo
  - bar

flags #

  • type: string[]
  • description: Array of strings, inspired by browser://flags, available values are determined by the specific implementation.
flags:
  - unlisted # https://indieweb.org/unlisted
  - draft

flags:
  - unlisted # https://indieweb.org/unlisted
  - draft

Extra #

Extra variables: This means that its naming is tentative and only used as a suggestion if needed.

audio #

  • type: string
  • description: Link to an audio file. u-audio[8]
audio: '/podcast.flac'

audio: '/podcast.flac'

video #

  • type: string
  • description: Link to an video file. u-video[9] or u-featured[6:1]
video: '/video.av1'

video: '/video.av1'

bookmark_of #

  • type: string
  • description: URL of specify the bookmark[10]. u-bookmark-of
bookmark_of: 'https://example.com/permalink47'

bookmark_of: 'https://example.com/permalink47'

like_of #

  • type: string
  • description: URL of specify the like[11]. u-like-of
like_of: 'https://example.com/permalink47'

like_of: 'https://example.com/permalink47'

repost_of #

  • type: string
  • description: For re-posting[12] other posts. u-repost-of
repost_of: 'https://example.com/permalink47'

repost_of: 'https://example.com/permalink47'

in_reply_to #

  • type: string | string[] (optional)
  • description: For replying[13] or multi-replying[14] to other posts. u-in-reply-to[15]
in_reply_to: 'https://example.com/post123'
# or
in_reply_to:
- 'https://example.com/post123'
- 'https://example.com/post456'

in_reply_to: 'https://example.com/post123'
# or
in_reply_to:
- 'https://example.com/post123'
- 'https://example.com/post456'

syndication #

  • type: string | string[]
  • description: URL(s) of syndicated copies of this post. u-syndication[16]
syndication: 'https://twitter.com/example/status/1542566710594658312'
# or
syndication:
- 'https://twitter.com/example/status/1542566710594658312'
- 'https://mastodon.social/@example/108964809799826911'

syndication: 'https://twitter.com/example/status/1542566710594658312'
# or
syndication:
- 'https://twitter.com/example/status/1542566710594658312'
- 'https://mastodon.social/@example/108964809799826911'
